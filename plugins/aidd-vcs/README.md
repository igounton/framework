# aidd-vcs

VCS workflow plugin for the AI-Driven Development framework.

Covers all external artifact creation: commits, pull/merge requests, release tags, and issue tracking.

## Skills

| Bracket ID | Skill | Description |
|---|---|---|
| [3.1] | [commit](skills/01-commit/README.md) | Create a git commit with proper conventional message format. |
| [3.2] | [pull-request](skills/02-pull-request/README.md) | Create PR (GitHub) or MR (GitLab) with filled template. |
| [3.3] | [release-tag](skills/03-release-tag/README.md) | Create and push a semantic version git tag with release notes. |
| [3.4] | [issue-create](skills/04-issue-create/README.md) | Create issues in the configured ticketing tool. |

## Assets

Each skill ships with VCS-specific templates in its `assets/` directory.
