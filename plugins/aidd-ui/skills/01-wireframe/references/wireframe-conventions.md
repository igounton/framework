# Wireframe conventions

How to draw the low-fidelity ASCII wireframes this skill produces.

- Box-drawing characters for regions; one box per screen.
- Number every region and give it a one-line note in a table beneath.
- Structure only: no color, no type, no styling, no behavior, no final copy — placeholders like `{...}` stand in for content.
