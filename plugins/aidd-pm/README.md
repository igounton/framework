# aidd-pm

Product management plugin for the AI-Driven Development framework.

> Status: release candidate. Skills and MCP configuration may change before stable release.

Covers ticket information retrieval, user story creation, product requirement documents, and spec generation for downstream agents.

## Skills

| Bracket ID | Skill | Description |
|---|---|---|
| [4.1] | ticket-info | Retrieve and display ticket information from the configured ticketing tool. |
| [4.2] | user-stories-create | Generate INVEST-compliant user stories from a feature description. |
| [4.3] | prd | Generate a structured Product Requirements Document. |
| [4.4] | spec | Generate and refine a project spec from a free-form human request. The spec is the immutable target a planner consumes. |
