# aidd-pm

Product management plugin for the AI-Driven Development framework.

> Status: release candidate. Skills and MCP configuration may change before stable release.

Covers ticket information retrieval, user story creation, and product requirement documents.

## Skills

| Bracket ID | Skill | Description |
|---|---|---|
| [4.1] | ticket-info | Retrieve and display ticket information from the project's configured ticketing tool. |
| [4.2] | user-stories-create | Generate well-structured user stories from feature requirements through iterative Product Owner questioning. |
| [4.3] | prd | Generate a Product Requirements Document from a feature description or user stories. |
