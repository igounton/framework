# Template JTBD (Jobs To Be Done)

## JTBD: [Need name]

**When** [observable situation]
**I want** [capability, not solution]
**So that** [functional objective]
**And feel** [emotional objective]

### Evidence

- Interview: [quote] — [source]
- Data: [measured metric]
- Observation: [observed behavior]

---

## Example

**JTBD: Client meeting preparation**

**When** I'm preparing an important client meeting
**I want** to have all project information quickly accessible
**So that** I can answer their questions without delay
**And feel** confident and professional in front of the client

### Evidence

- Interview: "I spend 30 min searching for info before each call" — Marie, Senior PM
- Data: 78% of PMs report wasting time searching for information (internal survey)
- Observation: Using 4+ different tools to prepare for a meeting

---

## Formulation rules

- The "**When**" must be a real, observable situation (not hypothetical).
- The "**I want**" expresses a user capability, not a technical solution.
- The "**So that**" is the measurable functional objective.
- The "**And feel**" captures the emotional objective (confidence, control, pride, serenity...).

---

## Anti-patterns

| Formulation | Problem | Correction |
| --- | --- | --- |
| "I want a button to..." | Technical solution, not capability | "I want to quickly access..." |
| "So that the system works" | Technical objective, not user | "So that I save time" |
| (no emotional dimension) | Missing deep motivation | Add "And feel..." |
| "When I click on..." | UI action, not real situation | "When I need to make a decision..." |
| "I want the app to be fast" | Too vague, not observable | "I want to get info in under 5 sec" |

---

## Validation checklist

- [ ] The "When" describes an observable situation (not a UI action)
- [ ] The "I want" expresses a capability (not a feature)
- [ ] The "So that" is a measurable benefit
- [ ] The "And feel" captures an authentic emotion
- [ ] At least 2 of 3 evidence types are documented
- [ ] No technical solution is presupposed
