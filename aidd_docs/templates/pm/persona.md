## VIBE Checklist

Before validating this persona, verify:

- [ ] **V**erify: Cited sources exist and are verifiable
- [ ] **I**nterpret: Profile is consistent with our product context
- [ ] **B**ias: Potential biases are identified (e.g., over-representation of a segment)
- [ ] **E**vidence: 2/3+ independent sources converge on key insights

## Best practices

### Generate all 3 variations

For each base persona, ask the AI to generate all 3 variations:

```text
From this persona [Name], generate 3 variations:
1. OPTIMIST version: enthusiastic early adopter
2. SKEPTIC version: reluctant user who compares to alternatives
3. POWER USER version: advanced user who pushes the limits

For each variation, adjust: motivations, frustrations, expectations, key quote.
```

### Avoid generic personas

| Anti-pattern | Problem | Solution |
| --- | --- | --- |
| "Marie, 35, executive" | Too generic, not actionable | Add specific context and verbatims |
| Persona without frustrations | Missing the "why" of the product | Document pain points with quotes |
| Persona without sources | Risk of hallucination | Require 2+ sources per insight |
| Single persona | Tunnel vision | Always create 2-3 distinct personas |
