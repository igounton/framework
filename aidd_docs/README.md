## Development Flow

### Quick flow

`/plan` → `/implement` → `/commit`

### Standard flow

`/plan` → `/isolate` → `/implement` + `/assert` + `/commit` → `/review_code` → `/create_request`

### Full flow (6 steps)

`/brainstorm` → `/new_issue` → `/plan` → `/isolate` → `/implement` + `/assert` + `/commit` → `/review_code` + `/review_functional` + `/create_request` → `/learn`
